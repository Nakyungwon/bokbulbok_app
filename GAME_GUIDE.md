# 복불복 게임 사용자 가이드 - 버전 3.0

## 🎮 빠른 시작

### 1단계: 참여하기

- 화면에 손가락을 올려놓고 **떼지 마세요**
- 원이 나타나면 참여 성공!
- **맥박 애니메이션**이 즉시 시작됩니다

### 2단계: 드래그하기

- 손가락을 움직이면 원이 따라다닙니다
- 원하는 위치로 자유롭게 이동 가능
- 맥박 애니메이션이 계속 유지됩니다

### 3단계: 게임 시작

- 참여자들이 준비되면 **자동으로 시작**
- 2.5초 카운트다운 후 당첨자 선정

### 4단계: 결과 확인

- 당첨자 원에서 특별한 애니메이션
- 진동으로 당첨 알림 (안드로이드)
- **당첨자가 아닌 원들은 모두 사라집니다**
- **당첨자 원만 고정 표시됩니다**

### 5단계: 역방향 애니메이션

- 당첨자 선정 후 **2.5초 뒤** 역방향 애니메이션
- 색상이 당첨자 원으로 모이는 효과

### 6단계: 자동 재시작

- 역방향 애니메이션 완료 후 **자동으로 새 게임 시작**
- 수동 버튼 없이 연속 플레이 가능

## 🎯 게임 규칙

### 참여 방법

- ✅ **올바른 방법**: 손가락을 올려놓고 떼지 않기
- ❌ **잘못된 방법**: 손가락을 떼거나 클릭하기

### 참여자 관리

- **최대 15명**까지 참여 가능
- **손가락을 떼면** 즉시 참여 취소
- **드래그 중에도** 참여 상태 유지
- **맥박 애니메이션**으로 생동감 표현

### 게임 진행

- **최소 1명** 이상 참여 시 게임 시작
- **마지막 이벤트** 후 즉시 시작
- **게임 진행 중**에는 새 참여 불가
- **자동 재시작**으로 연속 플레이

## 🎨 게임 특징

### 색상 시스템

- 각 참여자는 **고유한 색상** 할당
- 색상 중복 없음
- 15가지 미리 정의된 색상 + 랜덤 색상
- **어두운 테두리**로 원의 경계 명확화

### 애니메이션 효과

- **참여 시**: Elastic 애니메이션으로 원 등장
- **맥박**: 참여자 원이 생동감 있게 맥박치는 효과
- **드래그**: 실시간 위치 업데이트
- **당첨 시**: 퍼져나가는 승자 애니메이션
- **역방향**: 색상이 당첨자 원으로 모이는 효과

### 피드백 시스템

- **진동**: 안드로이드에서 0.2초 진동
- **시각적**: 당첨자 원 강조 표시
- **애니메이션**: 승자 효과 + 역방향 효과
- **맥박**: 지속적인 생동감 표현

## 🔧 문제 해결

### 진동이 느껴지지 않아요

1. **기기 확인**: 실제 안드로이드 기기인지 확인
2. **진동 설정**: 기기의 진동이 켜져 있는지 확인
3. **앱 권한**: 진동 권한이 허용되었는지 확인

### 게임이 시작되지 않아요

1. **참여자 수**: 최소 1명 이상 참여했는지 확인
2. **터치 방식**: 손가락을 떼지 않고 유지했는지 확인
3. **화면 터치**: 화면을 제대로 터치했는지 확인

### 애니메이션이 부드럽지 않아요

1. **기기 성능**: 다른 앱을 종료해보세요
2. **참여자 수**: 너무 많은 참여자는 성능에 영향
3. **브라우저**: 웹에서는 최신 브라우저 사용 권장

### 자동 재시작이 안 돼요

1. **역방향 애니메이션**: 2.5초 뒤 역방향 애니메이션이 시작되는지 확인
2. **애니메이션 완료**: 역방향 애니메이션이 완료될 때까지 기다리기
3. **게임 상태**: 게임이 정상적으로 종료되었는지 확인

## 💡 게임 팁

### 최적의 게임 경험

- **참여자 수**: 2-8명이 가장 재미있음
- **기기**: 실제 안드로이드 기기 사용 권장
- **환경**: 밝은 곳에서 플레이
- **맥박 애니메이션**: 게임의 생동감을 느껴보세요

### 전략적 플레이

- **위치 전략**: 다른 참여자와 거리를 두고 배치
- **드래그 활용**: 게임 시작 전 마지막 위치 조정
- **팀워크**: 친구들과 함께 즐기기
- **맥박 활용**: 맥박 애니메이션으로 게임 분위기 조성

### 재미있는 활용법

- **점심 메뉴**: 점심 메뉴 선택
- **순서 정하기**: 게임 순서 결정
- **벌칙 정하기**: 벌칙자 선정
- **팀 나누기**: 팀 구성
- **연속 게임**: 자동 재시작으로 빠른 연속 플레이

### 새로운 기능 활용

- **맥박 애니메이션**: 참여자 원의 생동감을 즐기세요
- **자동 재시작**: 수동 버튼 없이 연속 플레이
- **역방향 애니메이션**: 당첨자 선정 후의 특별한 효과
- **당첨자 고정 표시**: 당첨자가 명확하게 구분됩니다
