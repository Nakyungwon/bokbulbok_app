# 복불복 게임 사용자 가이드

## 🎮 빠른 시작

### 1단계: 참여하기

- 화면에 손가락을 올려놓고 **떼지 마세요**
- 원이 나타나면 참여 성공!

### 2단계: 드래그하기

- 손가락을 움직이면 원이 따라다닙니다
- 원하는 위치로 자유롭게 이동 가능

### 3단계: 게임 시작

- 참여자들이 준비되면 **자동으로 시작**
- 2초 카운트다운 후 당첨자 선정

### 4단계: 결과 확인

- 당첨자 원에서 특별한 애니메이션
- 진동으로 당첨 알림 (안드로이드)

---

## 🎯 게임 규칙

### 참여 방법

- ✅ **올바른 방법**: 손가락을 올려놓고 떼지 않기
- ❌ **잘못된 방법**: 손가락을 떼거나 클릭하기

### 참여자 관리

- **최대 15명**까지 참여 가능
- **손가락을 떼면** 즉시 참여 취소
- **드래그 중에도** 참여 상태 유지

### 게임 진행

- **최소 1명** 이상 참여 시 게임 시작
- **마지막 이벤트** 후 즉시 시작
- **게임 진행 중**에는 새 참여 불가

---

## 🎨 게임 특징

### 색상 시스템

- 각 참여자는 **고유한 색상** 할당
- 색상 중복 없음
- 15가지 미리 정의된 색상 + 랜덤 색상

### 애니메이션 효과

- **참여 시**: Elastic 애니메이션으로 원 등장
- **드래그**: 실시간 위치 업데이트
- **당첨 시**: 퍼져나가는 승자 애니메이션

### 피드백 시스템

- **진동**: 안드로이드에서 0.2초 진동
- **시각적**: 당첨자 원 강조 표시
- **애니메이션**: 승자 효과

## 🔧 문제 해결

### 진동이 느껴지지 않아요

1. **기기 확인**: 실제 안드로이드 기기인지 확인
2. **진동 설정**: 기기의 진동이 켜져 있는지 확인
3. **앱 권한**: 진동 권한이 허용되었는지 확인

### 게임이 시작되지 않아요

1. **참여자 수**: 최소 1명 이상 참여했는지 확인
2. **터치 방식**: 손가락을 떼지 않고 유지했는지 확인
3. **화면 터치**: 화면을 제대로 터치했는지 확인

### 애니메이션이 부드럽지 않아요

1. **기기 성능**: 다른 앱을 종료해보세요
2. **참여자 수**: 너무 많은 참여자는 성능에 영향
3. **브라우저**: 웹에서는 최신 브라우저 사용 권장

## 💡 게임 팁

### 최적의 게임 경험

- **참여자 수**: 2-8명이 가장 재미있음
- **기기**: 실제 안드로이드 기기 사용 권장
- **환경**: 밝은 곳에서 플레이

### 전략적 플레이

- **위치 전략**: 다른 참여자와 거리를 두고 배치
- **드래그 활용**: 게임 시작 전 마지막 위치 조정
- **팀워크**: 친구들과 함께 즐기기

### 재미있는 활용법

- **점심 메뉴**: 점심 메뉴 선택
- **순서 정하기**: 게임 순서 결정
- **벌칙 정하기**: 벌칙자 선정
- **팀 나누기**: 팀 구성
